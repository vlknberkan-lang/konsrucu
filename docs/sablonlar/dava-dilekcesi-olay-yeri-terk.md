## ALAN HARİTASI — her `{{alan}}` hangi kaynaktan dolar

| Alan | Kaynak | Çıkarım |
|---|---|---|
| `davaciAdres` `vekilAd` `vekilUets` | **Ayarlar / Şirket Bilgileri** (Ray + büro) | sabit (DB) |
| `davaliAd` `davaliTckn` | Hasar **Lehe formu** (rücu muhatabı) = İcra borçlu | regex (TCKN 11h) |
| `davaliAdres` | İcra **Ödeme Emri** (MERNİS adresi) | regex/metin |
| `icraDaire` `icraEsas` | İcra **Ödeme Emri** başlığı (“7. İcra Dairesi · 2026/798 Esas”) | regex |
| `davaDeger` (= asıl + işlemiş faiz) | İcra **Takip Talebi** “Toplam Alacak” | regex (tutar) |
| `sigortaliPlaka` | **Poliçe** (sigortalı araç) | regex (plaka) |
| `karsiPlaka` | İcra takip açıklaması / ekspertiz (zarar gören) | regex |
| `kazaTarihi` | Hasar **Lehe** (hasar tarihi) / takip açıklaması | regex (tarih) |
| `odemeTarihi` `odemeTutar` `odemeAlan` | Hasar **Dekont** (banka transfer) | regex |
| `policeNo` | **Poliçe** | regex |
| `hasarNo` `hasarSira` | Hasar **klasör adı / Lehe** | regex |
| `arabBuro` `arabBuroNo` `arabNo` | Açılmış davada **dava dilekçesi/delil dilekçesi** (numaralar); yeni üretimde **arabuluculuk son tutanağı** (dış) | regex / elle |
| `mahkemeIl` `mahkemeNo` | Yetki: sigortacı merkez/şube/acente **veya** kaza yeri (KTK 110/2); genelde İstanbul | kural + onay |
| `ihtiyatiHaciz` (var/yok) `ihtiyatiHacizGerekce` | **Karar** (davalı yaya/malvarlığı yok → var) | kullanıcı/asistan |
| `tarih` | dilekçe tarihi | sistem |

⚠ **Tek dış girdi — `arabuluculuk son tutanağı`:** İcra dosyasında **YOKTUR** (doğrulandı: arabuluculuk ayrı süreç → itiraz → Arabuluculuk Bürosu → dava). Açılmış davada dilekçeye **EK** olarak girer; numaralar dava/delil dilekçesinde geçer (regex). **Yeni dava üretirken** avukat tutanağı Arabuluculuk Bürosu'ndan alır → Büro No + Arabuluculuk No + anlaşamama tarihi oradan dolar.
